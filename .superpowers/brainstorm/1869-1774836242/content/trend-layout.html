Trend Chart Layout Options

How should longitudinal trends appear in the athlete profile panel?

A
Single Combined Chart Recommended

One line chart showing all 4 domain composite t-scores over time on a shared 0–100 scale. Each domain is a separate colored line. Compact — fits in the existing profile without major layout changes.

Domain T-Score Trends (0–100)
● CMJ ● GPS ● Body Wt ● Strength
Pros: Compact, shows cross-domain trajectory at a glance, easy to see if athlete is improving overall
Cons: Hides individual metric detail (e.g., which CMJ metric improved?)
B
4 Domain Mini Charts (2×2 Grid)

Each domain gets its own small chart showing its individual metrics as separate lines. More detailed but requires more vertical space below the existing profile content.

Cons: Takes significantly more vertical space, harder to compare across domains
C
Combined Chart + Click-to-Expand Detail

Default view shows the single combined t-score chart (Option A). Clicking a domain name expands it inline to show its individual metric lines. Best of both worlds — compact by default, detailed on demand.
